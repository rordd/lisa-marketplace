# pick — 랜덤 뽑기

## When to Use

- "저녁 메뉴 뽑아줘 — 김치찌개, 라면, 짜장"
- "누가 설거지 할지 정해줘 — 엄마, 아빠, 형"
- "오늘 운동 뭐 할까 — 헬스, 러닝, 요가, 수영"
- "3개 영화 골라줘 — 인터스텔라, 인셉션, 다크나이트, 테넷, 듄"

## When NOT to Use

- 확률 가중치 필요한 경우 (모든 후보 동일 확률)
- 정렬 / 우선순위 결정 (랜덤이라 매번 다름)

## Commands

```sh
cd skills/pick && sh scripts/pick.sh "<후보들>" [count]
```

후보 구분: 콤마(`,`) 또는 슬래시(`/`) 또는 줄바꿈. count 생략 시 1개.

예시:

```sh
sh scripts/pick.sh "김치찌개, 라면, 짜장면"          # 1개
sh scripts/pick.sh "엄마/아빠/형/누나/막내" 2         # 2개
sh scripts/pick.sh "헬스, 러닝, 요가, 수영, 자전거" 3 # 3개
```

출력 형식: 한 줄 한 후보씩.
